## 你好，我是 Jim

我叫万义尧（Jim Wan），是 Meta 的一名软件工程师，坐标纽约。白天，我在 MTIA 团队工作——确保 Meta 的广告模型在自研 AI 加速器上的数值精度。下班后，大概率在听 Japanese City Pop、打英雄联盟，或者和妻子一起 citywalk、打网球、做饭。

## 为什么要建这个网站？

其实做个人网站这个想法很久以前就有了，但一直没有行动。直到最近看到我的好朋友张熠（Yi Zhang）做了一个回顾职业历程的网站 [yizhang7210.com](https://yizhang7210.com/)，受此启发，我决定借助 AI 把这个拖了很久的想法付诸实践。

我想要的是：

- **一个属于自己的空间。** 不是 LinkedIn，不是 Twitter——一个完全由我控制的地方。
- **一个写作的地方。** 有些想法塞不进一条推文或一条 code review 评论。有的关于技术，有的关于生活，有的纯粹是碎碎念。
- **一种氛围。** 我想要温暖、平静的感觉。所以选了奶油色的设计，右下角还有一个 lofi BGM 播放器——如果你还没点那个音符按钮，试试看。

## 技术栈

如果你好奇的话，这个网站用了：

- **Hugo** — 一个快速的静态网站生成器，内置优秀的多语言支持
- **自定义主题** — 手工打造的毛玻璃设计，暖奶油色调
- **SCSS** 通过 Hugo Pipes 编译 — 零 Node.js 依赖
- **GitHub Pages** — 免费托管，通过 GitHub Actions 部署
- **双语** — 完整的中英文支持，因为我用两种语言思考

整个网站构建时间不到 200 毫秒。没有 JavaScript 框架，除了 Hugo 本身不需要任何构建工具。

## 接下来呢？

这里没有固定的主题。可能是工作中的小确幸，可能是职场成长的一些感悟，也可能是生活中的碎碎念——做了什么菜、去了哪里散步、打了一场酣畅淋漓的网球。有些会是技术性的，有些可能只有几句话。这就是碎碎念的意义所在。

感谢你的到来。

— Jim
